Leaf demo engine

Real-time graphics for fun and profit

About

Leaf is a graphics engine designed for demos. It is tightly integrated into Blender for efficient content creation.

The current version officially supports Blender 2.76b. It may work on other versions >= 2.74, but these are not supported.
